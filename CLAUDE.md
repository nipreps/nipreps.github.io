# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

NiPreps (NeuroImaging PREProcessing toolS) organization website and documentation, hosted at https://www.nipreps.org. Built with MkDocs + Material theme.

## Build & Development Commands

```bash
# Install dependencies
pip install -r requirements.txt

# Serve locally with live reload (primary development workflow)
mkdocs serve

# Build static site
mkdocs build

# Spell check (matches CI)
codespell

# Link check (after building)
mkdocs build --site-dir _site_preview
```

## Branch Model

- **`mkdocs`** — main/source branch (all edits go here)
- **`master`** — deployment branch (auto-generated by CI, never edit directly)

PRs target `mkdocs`, not `master`.

## CI/CD Workflows (`.github/workflows/`)

- **deploy.yml** — On push to `mkdocs`: generates member grid from GitHub API, builds site, deploys to `master` branch
- **codespell.yml** — Spell checking on push/PR to `mkdocs`
- **linkcheck.yml** — Validates internal/external links
- **update-telemetry.yml** — Automated telemetry chart updates (bot commits)

## Key Automation Scripts

- `scripts/generate_members_grid.py` — Queries GitHub API for org members, updates `docs/community/index.md` between marker comments. Requires `GITHUB_TOKEN`.
- `scripts/update_telemetry_images.py` — Updates telemetry chart references (`*_weekly.png`, `*_versionstream.png`) in `docs/index.md`.

## Documentation Content

All documentation lives under `docs/`. The nav structure is defined in `mkdocs.yml`. Key content areas:

- `docs/index.md` — Landing page with auto-updated telemetry charts
- `docs/intro/` — Framework architecture and transparency philosophy
- `docs/community/` — Members, contributing guidelines, code of conduct
- `docs/apps/` — BIDS-Apps execution guides (Docker, Singularity, DataLad)
- `docs/devs/` — Developer environment, versions matrix, releases
- `docs/assets/` — Images, logos, and auto-generated charts

## MkDocs Configuration Notes

- Material theme with custom fonts (Arimo text, Anonymous Pro code)
- Markdown extensions: admonition, tabbed content, code highlighting, superfences, snippets, emoji
- `markdown_include` extension with `base_path: docs` for file includes
- Custom CSS in `docs/stylesheets/extra.css`
